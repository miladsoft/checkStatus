Basic Usage

GET /api/status?url=YOUR_URL

Returns the status of a single website including response time and availability.

Example Response

{
  "url": "https://example.com",
  "status": 200,
  "responseTime": 123,
  "isUp": true,
  "lastChecked": "2023-09-20T12:00:00Z"
}

Response Fields

FieldDescription
urlThe URL that was checked
statusHTTP status code (200 = OK)
responseTimeResponse time in milliseconds
isUpBoolean indicating if the site is up
lastCheckedTimestamp of the last check
